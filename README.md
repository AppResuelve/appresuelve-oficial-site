# AppResuelve Website

Portfolio corporativo de [AppResuelve](https://appresuelve.com) — Soluciones Digitales a Medida.

Desarrollo web y móvil personalizado para empresas y emprendedores en Buenos Aires, Argentina.

## Stack

- **React** 19 + Vite
- **React Router** para routing
- **Framer Motion** para animaciones
- **pnpm** como package manager

## Proyectos

El sitio showcasea los productos de la empresa:

| Producto | Descripción | Stack |
|----------|-------------|-------|
| **WisePlace** | Plataforma de gestión educativa | PERN (PostgreSQL, Express, React, Node) |
| **FlavourLab** | Sistema para restaurantes | Docker, React, Node, PostgreSQL, AWS |
| **TengoTurno** | Sistema de turnos online | React, Node, MongoDB, PWA |

## Scripts

```bash
pnpm dev      # Desarrollo
pnpm build    # Build de producción
pnpm preview  # Preview del build
pnpm lint     # ESLint
pnpm format   # Prettier (formateo)
```

## Setup

```bash
pnpm install
pnpm dev
```
